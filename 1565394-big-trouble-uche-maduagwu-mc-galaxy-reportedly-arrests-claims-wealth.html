PR stunts in Nigeria lacks creativity

In a chat with Legit.ng, a PR consultant, Habib affirmed that PR stunts are strategies and can be the basis for virality if well thought out. He however, opined that strategies in Nigeria lack the creativity needed for the desired result.

In his words:

"Bad PR used to be PR anyway, but I don’t believe it works that way anymore. Stunts are supposed to be a strategy that is well thought out, a lot of that done in Nigeria lacks creativity and I’ll say there are quite a few Nigerian acts using publicity stunts the right way. PR stunts can be the basis for virality and are very effective if it’s well-thought-out and followed by an awesome project. If they are not planned and assessed properly before embarking on them, it’ll be futile."
